# 1. Created (Створено)

Замовлення сформоване, але ще не відправлене на оплату.

Подія переходу:

submit() → користувач натискає "Підтвердити".

# 2. PendingPayment (Очікує оплати)

Замовлення чекає успішної оплати.

Події переходів:

paymentSuccess() → оплата пройшла успішно.

paymentFailed() → оплата не пройшла → замовлення скасовується.

# 3. Paid (Оплачено)

Оплата пройшла, потрібно згенерувати квиток.

Подія:

generateTicket() → система створює електронний квиток.

# 4. TicketIssued (Квиток згенеровано)

Квиток сформований і готовий до відправки користувачу.

Подія:

deliver() → квиток відправляється користувачу.

# 5. Completed (Завершено)

Квиток доставлено, замовлення успішно завершене.

6. Cancelled (Скасовано)

Оплата не пройшла, або користувач відмінив замовлення.
